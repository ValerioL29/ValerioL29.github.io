题目：36. 有效的数独

难度：中等

遇见次数：第一次

涉及知识点：

  • 数组
  • 哈希表

提交

题解

一次遍历

首先，让我们来讨论下面两个问题：

  • 如何枚举子数独？

可以使用 box_index = (row / 3) * 3 + columns / 3，其中 / 是整数除法。

  • 如何确保行 / 列 / 子数独中没有重复项？

可以利用 value -> count 哈希映射来跟踪所有已经遇到的值。

现在，我们完成了这个算法的所有准备工作：

  • 遍历数独。
  • 检查看到每个单元格值是否已经在当前的行 / 列 / 子数独中出现过：
    • 如果出现重复，返回 false。
    • 如果没有，则保留此值以进行进一步跟踪。
  • 返回 true。
