题目：560. 和为K的子数组

难度：中等

遇见次数：第二次

涉及知识点：

  • 前缀和
  • 哈希表

提交

前缀和+哈希表优化

题解

前缀和+哈希表优化

思路及算法

  1. 每个元素对应一个 “前缀和”
  2. 遍历数组，根据当前 “前缀和”，在 map 中寻找「与之相减 == k」的历史前缀和
  3. 当前 “前缀和” 与历史前缀和，差分出一个子数组，该历史前缀和出现过 c 次，等价于当前项找到 c 个子数组求和等于 k
  4. 遍历过程中，c 不断加给 count，最后返回 count
